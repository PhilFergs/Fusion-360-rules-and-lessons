# Fusion 360 Add-ins (Release Packages)

This repo contains two ready-to-install zip packages:

- PhilsDesignTools-1.0.0.zip
  - Steel member generation + utilities (EA/SHS/RHS, rotate, rename, hole cut, stub arms, exports).
- PhilsBom.bundle-1.03.zip
  - BOM export add-in (CSV/XLSX/XML/JSON, unit control, include/exclude options, mass totals).

Each zip contains only the add-in folder and an Install_Instructions.txt file.

Install
1) Download and unzip the desired package.
2) Open Install_Instructions.txt inside the zip and follow the steps.
3) Restart Fusion 360 and run the add-in from Tools > Add-ins > Scripts and Add-ins.
